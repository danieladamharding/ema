Step 4 - Optional Settings
    Although Turnitin defines these settings as 'Optional', they are essential and should always be reviewed.
  • Enter special instructions: This should include helpful information relating to the assignment. Details entered here will appear to students underneath the Turnitin submission link.

    If marking anonymously, advising students to include their student ID as part of their assignment title will help to attribute marks when transfering them to the student records system.

  • Allow submissions after the due date: Setting this to 'Yes' will allow students with extensions or extenuating circumstances to still submit.
  • Originality report: Yes. The originality report generates a percentage score that suggests how much of an assignment matches the Turnitin database.
  • Generate Originality Reports for student submissions: On due date. This will compare all submissions together, once the due date passes.
  • Exclude bibliographic materials from Similarity Index for all papers in this assignment? No. Filtering can be applied later.
  • Exclude quoted materials from Similarity Index for all papers in this assignment? No. Filtering can be applied later.
  • Exclude small matches: No. Filtering can be applied later.
  • Allow students to see Originality Reports: No.

    It is advised that all students are given access to an originality report in their first assignment. Tutors should explain the way in which Turnitin is used at Keele.

  • Reveal grades to students only on post date: Yes. Please be aware that if set to 'No', all grades and feedback will be visible to students immediately.
  • Enable anonymous marking? Keele Regulation 8.5 states that work submitted for assessment that is worth more than 25% of the final mark should always be marked anonymously, unless you have received an exemption from the University Learning and Teaching Committee.
  • Submit papers to: Standard paper repository.

    You may wish to select 'No repository' if this Turnitin dropbox is intended for draft assignments or submissions you do not wish future documents to be matched against.

  • Search options: Select all checkboxes.
  • Attach a Rubric/form to this assignment: A rubric may be added here, although you may wish to apply it later, prior to marking.
  • Click the Submit button.

Search Options

The Search Options setting allows the instructor to select which Turnitin repositories students' submissions are checked against when processing Originality Reports.


The similarity index percentage may decrease if a repository option is deselected.

  • Student paper repository - Contains student papers from institutions using Turnitin from across the UK and internationally.
  • Institution paper repository - Only contains papers submitted as part of Keele University programmes.
  • Current and archived internet - As described, Turnitin searches live internet sites as well as an archive of 60 billion indexed web pages.
  • Periodicals, journals, & publications - Papers are matched against over 150 million academic documents, including journals, books and periodicals.

What is a rubric?

A rubric is a method of dividing the marking scheme into sections.

Rubrics are split into rows (the criteria) and columns (the value).

Tutors can click on each of the rubric's sections to award an appriopriate value for each criteria.

If using numerical values, the rubric can also be used to calculate an overall grade based on the selections.
