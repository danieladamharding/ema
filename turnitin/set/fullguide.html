Please refer to your School's local procedure before creating a new Turnitin assignment dropbox.
Step 1 - Creating the Turnitin Submission Link
  • Sign in to the Keele Learning Environment (KLE) and navigate to the module you wish to create a Turnitin assignment dropbox.
  • Ensure that Edit Mode is switched on in the top right corner of the browser window.
  • Choose a suitable location (e.g. a folder labelled Assessments) and select the Assessments menu. From the dropdown list, click Turnitin Assignments .
Step 2 - Choosing an Assignment Type

Three types of Turnitin assignments are available:

  • Paper assignment:
  • PeerMark assignment:
  • Revision assignment:
Step 3 - Assignment Details
  • Assignment title: Enter the name of the assignment (e.g. 1st Semester Final Essay).
  • Point value: The total amount of marks possible (usually 100).
  • Start date: The date and time students can begin submitting their assigments.
  • Due date: The deadline by which assignments are due.
  • Post date: The date that marks and feedback are released to students.
Editing Turnitin settings after students begin to submit their work is not recommended as it has been known to cause issues relating to the release of feedback.
Step 4 - Optional Settings
Although Turnitin defines these settings as 'Optional', they are essential and should always be reviewed.
  • Enter special instructions: This should include helpful information relating to the assignment. Details entered here will appear underneath the Turnitin submission link.

    If marking anonymously, advising students to include their student ID as part of their assignment title will help to attribute marks when transfering them to the students record system.

  • Allow submissions after the due date: Setting this to 'Yes' will allow students with extensions or extenuating circumstances to still submit.
  • Originality report: Yes. The originality report generates a percentage score that suggests how much of an assignment matches the Turnitin database.
  • Generate Originality Reports for student submissions: On due date. This will compare all submissions together, once the due date passes.
  • Exclude bibliographic materials from Similarity Index for all papers in this assignment? No. Filtering can be applied later.
  • Exclude quoted materials from Similarity Index for all papers in this assignment? No. Filtering can be applied later.
  • Exclude small matches: No. Filtering can be applied later.
  • Allow students to see Originality Reports: No.

    It is advised that all students are given access to an originality report in their first assignment. Tutors should explain the way in which Turnitin is used at Keele.

  • Reveal grades to students only on post date: Yes. Please be aware that if set to 'No', all grades and feedback will be visible to students immediately.
  • Enable anonymous marking? Keele Regulation 8.5 states that work submitted for assessment that is worth more than 25% of the final mark should always be marked anonymously, unless you have received an exemption from the University Learning and Teaching Committee.
  • Submit papers to: Standard paper repository.

    You may wish to select 'No repository' if this Turnitin dropbox is intended for draft assignments or submissions you do not wish future documents to be matched against.

  • Search options: Select all checkboxes.
  • Attach a Rubric/form to this assignment: A rubric may be added here, although you may wish to apply it later, prior to marking.
  • Click the Submit button.
Step 5 - Roster Sync
  • When using Turnitin to view student submissions you may occasionally notice that not every student is listed in the Turnitin dropbox, if this happens it can be fixed easily.

  • Please note that if students are missing from the Turnitin inbox, it does not stop them submitting. Any student enrolled on the module can still submit, but their submission will not appear until the Roster Sync button has been clicked.

  • From within the Turnitin inbox, select Roster Sync from the options available on the right hand side.
  • Clicking Roster Sync will force Turnitin to re-load the list of students. Any missing students will now be added back in.
  • Once the brief message disappears, all of your students should be loaded into the Turnitin dropbox.
Well done! You have successfully created a Turnitin assignment dropbox.

Edit Mode

Edit Mode allows Instructors to make changes to content and toggles on and off. The toggle appears in the navigation bar in the top right corner of the screen.

To edit content, toggle Edit Mode to On. To view course content as a student, toggle Edit Mode to Off.

Assessment Section

If you have used one of the KLE templates, you will already have an Assessments section.You will find it on the left hand navigation menu.

If you have not used a KLE template, you will need to add a folder or section for your assessments.

Paper Assignment

Paper Assignment: The paper assignment type is the base assignment for all other assignments types (PeerMark, Revision, and Reflection).

When creating a paper assignment there are three dates that instructors will need to set: the start date, the due date, and the post date.

PeerMark Assignment

The PeerMark assignment type allows students to review their peers' papers based on scale and free response questions selected by the instructor.

Revision Assignment

If the instructor would like students to submit multiple drafts without overwriting the previous drafts/submissions, the instructor can create additional assignments using the revision assignment type.

Revision assignments are duplicates of the 'parent' assignment's advanced options and standard settings, but may have new start, due, and post dates.
