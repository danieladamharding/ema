Please refer to your School's local procedure before creating a new Turnitin assignment dropbox.
Step 1 - Creating the Turnitin Submission Link
  • Sign in to the Keele Learning Environment (KLE) and navigate to the module you wish to create a Turnitin assignment dropbox.
  • Ensure that Edit Mode is switched on in the top right corner of the browser window.
  • Choose a suitable location (e.g. a folder labelled Assessments) and select the Assessments menu. From the dropdown list, click Turnitin Assignments.
Next Step

Edit Mode

Edit Mode allows Instructors to make changes to content and can be toggled on and off. The toggle appears in the navigation bar in the top right corner of the screen.


To edit course content, toggle Edit Mode to ON. Switching Edit Mode to OFF will provide a basic preview of how course content will appear to students.

Assessments Section

If you have used one of the KLE templates, you will already have an Assessments section.You will find it in the left hand navigation menu. This provides a clearly defined space for students to receive information about their assessments and to submit.


If you have not used a KLE template, it is recommended that you add a similar folder for assessments.
