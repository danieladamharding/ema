Please refer to your School's local procedure before creating a new Turnitin assignment dropbox.
Step 1 - Creating the Turnitin Submission Link
  • Sign in to the Keele Learning Environment (KLE) and navigate to the module you wish to create a Turnitin assignment dropbox.
  • Ensure that Edit Mode is switched on in the top right corner of the browser window.
  • Choose a suitable location (e.g. a folder labelled Assessments) and select the Assessments menu. From the dropdown list, click Turnitin Assignments .
Next Step

Edit Mode

Edit Mode allows Instructors to make changes to content and toggles on and off. The toggle appears in the navigation bar in the top right corner of the screen.

To edit content, toggle Edit Mode to On. To view course content as a student, toggle Edit Mode to Off.

Assessment Section

If you have used one of the KLE templates, you will already have an Assessments section.You will find it on the left hand navigation menu.

If you have not used a KLE template, you will need to add a folder or section for your assessments.
