  1. Author: Once anonymous marking is enabled, a grey box will obscure student names. Once a submission has been received this cannot be turned off for the whole class, only individually. After the post date has passed (when marks and feedback are released), student names will re-appear.
  2. Title: This is the title of the student’s submission and should include their Keele ID number. However, it is the student’s responsibility to enter this title at the point of submission.
  3. Similarity: This provides a % text match, indicating how much of the assignment matches sources stored within the Turnitin database. Turnitin uses a RAG rating (Red, Amber, Green) according to the percentage of matched content. To establish plagiarism, further investigation will be required and should not be based on this score alone.
  4. Grade: Displays the grade that has been awarded within the Turnitin document viewer.
  5. Response: If a student has viewed their feedback for 30 seconds or more, a silhouette icon will appear here.
  6. File: If anonymous marking is enabled, clicking the file icon after the post date has passed will allow an instructor to download the originally submitted file.
  7. Paper ID: Each submission is given a unique ID number. The Paper ID will match the digital receipt provided to students after submission and can be useful for troubleshooting purposes.
  8. Date: Displays the date each student submitted their assignment. The date will appear in red if it was submitted after the due date.
