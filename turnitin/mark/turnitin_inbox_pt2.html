Understanding the Turnitin Inbox (2 of 3)
Displays information about the submission including file size, word count, page count etc. It will also show how many comments/QuickMarks have been added to the paper.
Generates a unique code that enables marking using the Turnitin iPad app.
Download the originally submitted file or as a PDF document (which can then be printed).
The zoom slider can be used to adjust the magnification of the document viewer to increase/decrease the size of the page.

    Hover over the highlighted areas above to reveal their function.
  1. Author: Once anonymous marking is enabled, a grey box will prevent each student's name appearing. Once a submission has been received this cannot be turned off for a whole class, only individually. Once the post date has passed, students names will re-appear.
  2. Title: This is the title of the student’s submission and should include their Keele ID number. However, it is the student’s responsibility to enter this title at the point of submission.
  3. Similarity: This provides a % text match, indicating how much of the assignment matches sources from sources stored within the Turnitin database. Turnitin uses a RAG rating (Red, Amber, Green) according to the percentage of matched content, however, this should be treated with caution and is not an indicator of plagiarism in itself.
  4. Grade: Displays the grade that has been awarded within the Turnitin document viewer.
  5. Response: If a student has viewed their feedback for 30 seconds or more, a silhouette icon will appear here.
  6. File: If anonymous marking is enabled, clicking the file icon after the post date has passed will allow an instructor to download the originally submitted file.
  7. Paper ID: Each submission is given a unique ID number. The Paper ID will match the digital receipt provided to students after submission and can be useful for troubleshooting purposes.
  8. Date: Displays the date each student submitted their assignment. The date will appear in red if it was submitted after the due date.
