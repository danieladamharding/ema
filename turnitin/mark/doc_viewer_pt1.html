Understanding the Turnitin Document Viewer (1)

Image too small? Click here to enlarge.

  • Author: Once anonymous marking is enabled, a grey box will prevent each student's name appearing. Once a submission has been received this cannot be turned off for a whole class, only individually. Once the post date has passed, students names will re-appear.
  • Title: This is the title of the student’s submission and should include their Keele ID number. However, it is the student’s responsibility to enter this title at the point of submission.
  • Similarity: This provides a % text match, indicating how much of the assignment matches sources from sources stored within the Turnitin database. Turnitin uses a RAG rating (Red, Amber, Green) according to the percentage of matched content, however, this should be treated with caution and is not an indicator of plagiarism in itself.
  • Grade: Displays the grade that has been awarded within the Turnitin document viewer.
  • Response: If a student has viewed their feedback for 30 seconds or more, a silhouette icon will appear here.
  • File: If anonymous marking is enabled, clicking the file icon after the post date has passed will allow an instructor to download the originally submitted file.
  • Paper ID: Each submission is given a unique ID number. The Paper ID will match the digital receipt provided to students after submission and can be useful for troubleshooting purposes.
  • Date: Displays the date each student submitted their assignment. The date will appear in red if it was submitted after the due date.
