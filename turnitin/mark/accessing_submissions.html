Accessing Student Submissions
  • Sign in to the Keele Learning Environment (KLE) and navigate to the module you wish to access student submissions for.
  • From the Course Management menu on the left side of the screen, select Course Tools.
  • From the list of options available, select Turnitin assignments.
  • Select the appropriate Turnitin dropbox. If you have created multiple assignments, choose from the list.
  • You should now see the Turnitin inbox. The inbox displays all of those who are due to submit (or have already submitted) an assignment.
