The GradeMark app is only available for iPad devices.
Step 1 - Setting up the app
  • From a desktop PC or Mac, sign in to the Keele Learning Environment (KLE) and navigate to the module which contains the Turnitin inbox you wish to mark student submissions for. If you are unsure how to do this, please see the guide at https://www.keele.ac.uk/turnitin/accessingsubmissions.
  • From the Turnitin inbox, open any submission using the pencil icon ().
  • Located in the bottom left-hand corner of the document viewer, click on the iPad icon () and select Get Class Code for iPad.
  • A popup will appear. Click the Generate code button.
  • A sixteen character access code will be generated. Keep a note of this code, it will be required to access assignments on your iPad.
Step 2 - Signing into the Turnitin Feedback Studio app
  • If you have not already done so, download the Turnitin Feedback Studio app from the Apple iTunes App Store.
  • Launch the Feedback Studio app.
  • Tap the Log in with Access Code button and enter the 16 characters generated from the KLE (ensure the code has no spaces).
  • Tap Log in.
Step 3 - Module selection
  • Under Accounts, select University of Keele.
  • Tap the required module from the Classes list.
Step 4 - Synchronising with Feedback Studio
    It is recommended that submissions are synchronised. This updates the app to retrieve the most up-to-date information from Turnitin.
  • Select the required assignment.
  • To synchronise the dropbox, tap the Information icon and turn on Sync Submissions.
  • Assignments will now reside on the iPad with the app interface resembling the desktop inbox, allowing tutors to mark offline where an internet connection is not available.
