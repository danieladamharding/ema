Understanding the Turnitin Document Viewer (2 of 3)
Displays information about the submission including file size, word count, page count etc. It will also show how many comments/QuickMarks have been added to the paper.
Generates a unique code that enables marking using the Turnitin iPad app.
Download a converted PDF of the submission, its digital receipt or the originally submitted file format (once the post date has passed).
The zoom slider can be used to adjust the magnification of the document viewer to increase/decrease the size of the page.

    Hover over the highlighted areas above to reveal their function.
  1. Submission information: Displays information about the submission including file size, word count, page count etc. It will also show how many comments/QuickMarks have been added to the paper.
  2. Class access code for iPad: Generates a unique code that enables marking using the Turnitin iPad app.
  3. Print/download: Download a converted PDF of the submission, its digital receipt or the originally submitted file format (once the post date has passed).
  4. Magification slider: The slider can be used to adjust the document viewer and increase/decrease the size of the page.
